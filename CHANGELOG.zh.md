# 更新日志

### 1.0.0 (2025-06-05)

我们在 0.7 版本经过了长期且广泛的验证及应用, 现在决定将版本号正式转为 1.0 正式版. 

请注意本版本相较于 0.7.2 没有明显的变化, 您可以从 0.7 平滑过渡到此版本.

- 更新配置中的帮助界面显示选项
- 一些边缘修复

---

### 0.7.2 (2025-04-11)

- 简化装饰器写法: 使用 `@cli` 替代 `@cli.cmd()`

### 0.7.1 (2025-02-19)

- 修复对 `sys.argv` 的解析
- 修复对无参函数的调用

- 非侵入式调用脚本
